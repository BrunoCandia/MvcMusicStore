# ASP.NET MVC MUSIC STORE
*C# / Razor / Code First / *

**Requirements** (Visual Studio 2012, Sql Server, Entity Framework)

**Installation** 
- Open \MvcMusicStore\MvcMusicStore.sln in Visual Studio and Run. 
- Database and seed data will be created automatically.
